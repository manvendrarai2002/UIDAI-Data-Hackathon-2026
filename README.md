# Aadhaar Societal Trends Analysis 2025
Final submission for the UIDAI Data Hackathon.

## Project Structure
- **/notebooks**: Main analysis and visualization code.
- **/results**: Generated charts and insights.
- **/scripts**: Helper Python files for data cleaning.
- **/data**: Raw CSV files (not included in repository).

## Key Insights
1. **Saturation:** Updates now exceed enrolments by 10:1.
2. **Age Demographic:** 76% of new enrolments are minors (0-17).
3. **Q4 Surge:** Biometric updates spike significantly at year-end.

## Tech Stack
Python (Pandas, Matplotlib, Seaborn)

## How to Run
1. Place all Aadhaar CSV files in the `/data` folder
2. Navigate to `/notebooks` and open `Aadhaar_Analysis_2025.ipynb`
3. Run all cells sequentially
4. Check `/results` for generated visualizations
